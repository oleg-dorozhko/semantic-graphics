# Semantic Graphics

Стандарт семантичного малювання, де зображення створюється не пікселями, а атомарними об’єктами та явними взаємодіями.

## Що це таке
- Зображення = граф об’єктів (рука, прапор, атом, інструмент, AI).
- Зв’язки = взаємодії (“тримати”, “стискати”, “підтримувати”, “переносити”).
- Художник малює словами, а не пензлем.

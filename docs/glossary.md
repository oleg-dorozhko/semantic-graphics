# Словник Semantic Graphics

**Семантичний атом** — мінімальний об’єкт (рука, долоня, древко, атом, інструмент).  
**Взаємодія** — зв’язок: "тримає", "штовхає", "скручує".  
**Агент** — об’єкт, який виконує дію.  
**Пацієнт** — об’єкт, над яким діють.  
**Генеративний контур** — автоматично сформована частина (пальці → на древко).  
**Примітив** — базова форма для візуалізації об’єкта.
